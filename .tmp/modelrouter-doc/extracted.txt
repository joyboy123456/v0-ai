Model Router API 完整文档说明
概述
Model Router 是一个统一的大模型服务管理平台，提供 126 个模型的统一 API 访问。本文档基于实际测试结果编写，包含所有模型的分类、接口说明和调用示例。
基础信息
Base URL: https://model-router.edu-aliyun.com/v1
认证方式: Bearer Token（API Key）
公共请求头:
Authorization: Bearer <your-api-key>
Content-Type: application/json

API 接口说明
1. 文本对话 API
接口地址: POST /v1/chat/completions
兼容协议: OpenAI Chat Completions
请求参数
参数
类型
必填
说明
model
string
是
模型标识，如 qwen/qwen3-max
messages
array
是
对话消息列表，包含 role 和 content
stream
boolean
否
是否启用流式响应，默认 false
temperature
number
否
采样温度，范围 0-2
max_tokens
integer
否
最大生成 Token 数
enable_thinking
boolean
否
启用深度思考/思维链模式
注意事项
qwq 系列模型（qwq-plus 等）仅支持 stream 模式，需设置 "stream": true
多模态模型（视觉理解、OCR 等）需要在 messages 中传入图片
请求示例
curl https://model-router.edu-aliyun.com/v1/chat/completions \  -H "Content-Type: application/json" \  -H "Authorization: Bearer sk-xxx" \  -d '{    "model": "qwen/qwen3-max",    "messages": [{"role": "user", "content": "Hello!"}],    "stream": false  }'
2. 图片生成/编辑 API
接口地址: POST /v1/images/generations
兼容协议: OpenAI Images Generations
重要说明
新版模型（wan2.7-image-pro、wan2.7-image、wan2.6-image、wan2.6-t2i）支持同步调用
旧版模型（wan2.5-t2i-preview、wan2.5-i2i-preview、wan2.2-t2i-flash）仅支持异步调用，需要：
添加请求头：X-DashScope-Async: enable
使用 DashScope 原生格式：{"input": {"prompt": "...", "images": [...]}}
通过 /v1/tasks/{task_id} 查询任务结果
同步调用示例（新版模型）
curl https://model-router.edu-aliyun.com/v1/images/generations \  -H "Content-Type: application/json" \  -H "Authorization: Bearer sk-xxx" \  -d '{    "model": "qwen/wan2.7-image",    "prompt": "一只在星空下奔跑的猫",    "n": 1,    "size": "1024*1024"  }'
异步调用示例（旧版模型）
# 步骤1: 创建异步任务curl https://model-router.edu-aliyun.com/v1/images/generations \  -H "Content-Type: application/json" \  -H "Authorization: Bearer sk-xxx" \  -H "X-DashScope-Async: enable" \  -d '{    "model": "qwen/wan2.5-t2i-preview",    "input": {"prompt": "一间有着精致窗户的花店"},    "parameters": {"size": "1024*1024", "n": 1}  }'# 步骤2: 查询任务结果（使用返回的 task_id）curl https://model-router.edu-aliyun.com/v1/tasks/{task_id} \  -H "Authorization: Bearer sk-xxx"
图片编辑调用示例
curl https://model-router.edu-aliyun.com/v1/images/generations \  -H "Content-Type: application/json" \  -H "Authorization: Bearer sk-xxx" \  -H "X-DashScope-Async: enable" \  -d '{    "model": "qwen/wan2.5-i2i-preview",    "input": {      "prompt": "将风格改为油画",      "images": ["https://example.com/source.jpg"]    },    "parameters": {"size": "1024*1024"}  }'

常见问题
qwq 系列模型调用失败
原因：仅支持流式输出解决：设置 "stream": true
图片生成报 "url error"
原因：旧版模型（wan2.5/2.2）不支持同步调用解决：改用异步调用，添加 X-DashScope-Async: enable 请求头，使用 input.prompt 格式
向量/排序模型调用失败
原因：多模态模型需要特定输入格式解决：参照文档中的 JSON 格式示例
TTS 调用失败
原因：音色名称不匹配解决：使用 Chelsie、Ethan、Serena 等可用音色
