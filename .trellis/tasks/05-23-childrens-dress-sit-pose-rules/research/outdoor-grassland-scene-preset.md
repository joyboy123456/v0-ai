# 户外蓝天草地场景预设（用户提供 JSON）

> 来源：用户在 2026-05-23 brainstorm 第二轮对话直接给出的 JSON 预设。
> 用途：童装连衣裙 9 shot 裂变中，2/9 张图的固定外景常量（剩余 7/9 张沿用参考图原背景）。
> 本文件是 prompt 字段直接来源，写代码时按本表抄录中文措辞即可。

## 用途定位

`sunny_grassland_blue_sky_park_scene` — 固定户外草地蓝天场景，可用于儿童写真、亲子写真、时尚穿搭、生活方式类生图。

## 核心视觉常量（写进 prompt 用）

### 背景
- 明亮晴朗的户外公园草地场景
- 大片干净的蓝色天空作为背景
- 低地平线构图（天空留白充足）
- 远处有几棵稀疏的小树苗
- 背景轻微虚化（浅景深）
- 最少化背景元素（无杂物）

### 地面
- 自然绿色草坪
- 草地纹理自然微起伏
- 草地上有自然阳光阴影

### 氛围
- 清新、明亮、干净
- 春夏季感
- 户外生活方式的愉悦心情

## 光线

- 类型：自然日光
- 时间：晴朗白天
- 质感：明亮、干净、柔和阳光
- 阴影：草地上有可见自然阴影
- 色温：略偏暖的日光

## 相机 / 构图

- 方向：竖版人像
- 镜头类型：全身人像
- 角度：轻微低机位、温和仰拍
- 镜头感：35mm–50mm 真实人像镜头
- 景深：中等浅景深、背景轻微虚化
- 构图：主体居中偏下、低地平线、大片天空留白

## 视觉风格

- 视觉风格：真实摄影
- 画质：高分辨率、主体锐利、细节干净
- 色彩：明亮自然色彩、清新蓝绿调
- 心情：阳光、纯真、清新、活泼、户外生活方式

## 落地约束（本小姐写代码时要注意）

1. **场景二分策略**：9 shot 中固定 **2 张**走本预设外景常量，**7 张**沿用参考图原背景。
2. **哪 2 个 shot 走外景**：候选 `shot_3 左侧 60°` + `shot_9 自由互动`（侧身大角度 + 动态飘扬最适合大片天空 + 草地动态感），待与用户确认。
3. **白色金属折叠椅 shot 不走外景**：坐姿 shot_7 留在原背景，避免户外+椅子组合显得诡异。
4. **措辞 100% 中文**：本仓库其它 prompt 模板全中文，外景常量也走中文版以保持一致。
5. **不写入参考图槽位**：用户明确要求"我们不用加参考图路径，我们就做提示词工程就行"。
